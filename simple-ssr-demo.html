üîß How Summon SSR Works:
  • Server Rendering: PlatformRenderer converts @Composable functions to HTML
  • Component Based: Use familiar components like Column, Text, Button
  • Type Safety: Kotlin's type system prevents runtime errors
  • SEO Friendly: Complete HTML is generated server-side
  • Fast Loading: No client-side JavaScript required for initial render
üìã Example Summon SSR Code: 
@Composable
fun TodoApp(todos: List<Todo>) {
    Column {
        Text("üìù Summon SSR Todo App")
        todos.forEach { todo ->
            TodoItem(todo = todo)
        }
    }
}

// In your server route:
val html = renderer.renderComposableRoot { 
    TodoApp(todos = todoStorage.getAllTodos()) 
}
call.respondText(html, ContentType.Text.Html)
